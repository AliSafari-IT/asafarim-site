### Add to existing project

To use this package in an existing project:

```bash
yarn add @fluentui/react
```

Then you can import it to your React javascript or typescript file as follow:

```
import { DesiredFluentUiComponentName } from '@fluentui/react';

# example:
import { Checkbox, Stack } from "@fluentui/react";
```

### Explore Fluent UI controls

Fluent UI's  web controls are built with the React framework. [Here](https://developer.microsoft.com/en-us/fluentui#/controls/web) you can look through the control list available in Fluent UI React.
